Features

creates a bar on top of each editor for manage tabs, with a button for sidebar toggle, and another to see a list of tabs
  • drag & drop
  • drag & drop path
  • button flip view
  • button close file (include middle click)
  • information about file in tooltip
  • It contains icons of your choice
  • enable opacity of the panel
  • icons in mode gray scale
  • multiple buttons (tabmenu)
also has an option to display the sidebar when the mouse is over the element, and finally a button on the status bar to toggle the toolbar.more info,Report a Bug
